> 발표 시간: 약 12분
> 대상: 비개발자, 일반 청중

## 3. 상세 시연 시나리오

### 3.1 도입부 (1분 30초)

**멘트**:
> "안녕하세요, D106 팀장 윤상훈입니다. 흩어진 공부를 모으고, 잊혀질 지식을 퀴즈로 묶다: AI 올인원 스터디 플랫폼 SQUIZ를 소개드리겠습니다."

**클릭 순서**: (화면 없음, 슬라이드)

1. 타이틀 슬라이드 표시
2. "스터디 해보신 분 손 들어주세요" (청중 참여)
3. "한 달 넘게 유지된 분?" (손 줄어듦)
4. 문제 슬라이드: 파편화된 도구, 관리 체계 부재, 성장 안 보임

---

### 3.2 대시보드 소개 (30초)

**멘트**:
> "SQUIZ에 로그인하면 이 화면이 맞이합니다. 오늘 풀어야 할 복습 퀴즈, 다가오는 스터디 일정, 지난 모임 요약까지. 고민할 필요 없이 바로 시작하면 됩니다."

**클릭 순서**:

1. 로그인 상태로 대시보드 진입
2. **왼쪽 상단**: 오늘의 복습 퀴즈 위젯 가리키기
3. **오른쪽**: 다가오는 일정 위젯 가리키기
4. **하단**: 내 스터디 목록 간단히 소개

---

### 3.3 AI 학습 계획 - 커리큘럼 생성 (45초)

**멘트**:
> "스터디를 만들 때 가장 큰 고민, '무엇을 공부하지?'입니다. 주제만 입력하면 AI가 몇 초 만에 회차별 학습 계획을 만들어줍니다."

**클릭 순서**:

1. 상단 메뉴 → **[스터디 만들기]** 클릭
2. 스터디 정보 입력 (이름: "자바 백엔드 마스터")
3. **[AI 커리큘럼 생성]** 버튼 클릭
4. 주제 입력: "자바 백엔드"
5. AI가 실시간으로 8주 커리큘럼 생성되는 것 확인
6. 생성된 커리큘럼 간단히 스크롤
7. (선택) 기존 스터디 → 로드맵 보기 → 자동차 현재 위치 확인

---

### 3.4 워크스페이스 + 캘린더 (45초)

**멘트**:
> "스터디 안에서 바로 채팅하고, 자료를 공유할 수 있습니다. 카톡에서 자료 찾으려고 스크롤 올릴 필요 없습니다."

**클릭 순서**:

1. 기존 스터디 클릭 → 워크스페이스 진입
2. **채팅 영역**: 메시지 확인, 핀 고정된 메시지 표시
3. **자료실 탭** 클릭 → 공유된 파일 목록 확인
4. **캘린더 탭** 클릭 → 통합 캘린더 표시
5. 개인 일정 + 스터디 일정이 함께 표시되는 것 강조
6. Google Calendar 아이콘 → 연동 설명

---

### 3.5 화상 모임 + AI 자동 정리 (45초)

**멘트**:
> "화상 모임이 끝나면 AI가 자동으로 요약해줍니다. 회의록 쓰느라 모임에 집중 못하던 일, 이제 없습니다."

**클릭 순서**:

1. 스터디 → **[미팅]** 탭 클릭
2. 이전 미팅 기록 목록에서 하나 선택
3. **AI 요약 결과** 확인:
   - 핵심 내용 요약
   - 오개념 교정
   - 심화 학습 추천
   - 다음에 할 일 목록
4. **[PDF 저장]** 또는 **[마크다운 저장]** 버튼 표시
5. (선택) 실제 화상 모임 화면 캡처/영상으로 대체

---

### 3.6 퀴즈 학습 시스템 (1분 30초)

**멘트**:
> "사람은 배운 걸 24시간 안에 70%를 잊어버립니다. SQUIZ는 잊어버리기 직전에 AI가 다시 문제를 내줍니다."

**클릭 순서**:

#### A. 매일 복습 퀴즈

1. 상단 메뉴 → **[퀴즈]** 클릭
2. **[오늘의 복습]** 카드 클릭
3. 복습 퀴즈 1~2문제 시연 (정답/오답 표시)
4. "틀린 문제는 더 자주, 맞힌 문제는 덜 자주" 설명

#### B. 약점 분석

5. **[약점 분석]** 탭 클릭
6. 분야별 정답률 차트 표시
7. "이 분야를 더 공부하세요" 추천 카드 강조

#### C. 코스 학습

8. **[코스]** 탭 클릭
9. 운영체제/네트워크/데이터베이스 등 코스 목록 표시
10. 하나의 코스 진행률 확인

#### D. 오답 노트

11. **[오답 노트]** 탭 클릭
12. 틀린 문제 목록 확인

---

### 3.7 모바일 앱 - 블루투스 출석 (1분 30초)

**멘트**:
> "SQUIZ는 앱으로도 쓸 수 있습니다. 오프라인 스터디에서 앱으로만 가능한 특별한 기능이 있습니다."

**준비물**: 안드로이드 폰 2대 (스터디장용 + 스터디원용), 화면 미러링

**클릭 순서**:

#### A. 앱 소개

1. 앱 홈 화면 표시 (대시보드)
2. 하단 탭: 홈 / 스터디 / 퀴즈 / 마이페이지 간단히 소개

#### B. 블루투스 출석 시연

3. **스터디장 폰**: 스터디 → 출석 관리 → **[출석 시작]** 버튼 클릭
4. "블루투스 출석이 시작되었습니다" 알림 표시
5. **스터디원 폰**: 스터디장 폰 근처(약 10m)에 위치
6. 스터디원 폰에서 자동으로 출석 인식 알림 표시
7. 스터디장 폰: 출석 현황에 스터디원 이름 표시

**멘트**:
> "명단 돌리거나 일일이 체크할 필요 없습니다. 블루투스 기술로 실내에서도 정확하고, 배터리 소모도 거의 없습니다."

#### C. 푸시 알림 + 녹음

8. 푸시 알림 예시 화면 (스터디 시작 10분 전 알림)
9. 녹음 기능: 오프라인 모임 중 녹음 → AI 자동 정리 연동 설명

---

### 3.8 게이미피케이션 (30초)

**멘트**:
> "매일 퀴즈를 풀고, 모임에 참여하면 경험치가 쌓입니다. 레벨이 오르면 귀여운 돌고래가 축하해줍니다."

**클릭 순서**:

1. 마이페이지 → 프로필 진입
2. 레벨, 경험치 바 표시
3. 뱃지 컬렉션 표시
4. 활동 기록 (GitHub 잔디 스타일) 히트맵 표시
5. (선택) 레벨업 축하 애니메이션 재생

---

### 3.9 비교 + 정리 (30초)

**멘트**:
> "정리하면 이렇습니다. 기존에는 카톡, 줌, 노션을 따로 써야 했습니다. SQUIZ는 이 모든 걸 하나로 통합하고, AI와 동기부여 시스템까지 더했습니다."

**클릭 순서**: (슬라이드)

1. 비교표 슬라이드 표시
   - 카톡+줌+노션 vs SQUIZ
   - 올인원 / AI / 동기부여 강조

---

### 3.10 클로징 (30초)

**멘트**:
> "카톡, 줌, 노션을 따로 쓰는 시대는 끝났습니다. SQUIZ 하나면 충분합니다. AI가 계획하고, 기록하고, 복습시켜주는 — 학습의 동반자, SQUIZ. 감사합니다."

**클릭 순서**:

1. 클로징 슬라이드: 웹 + 앱 화면 나란히 배치
2. QR 코드 (서비스 URL) 표시
3. 팀원 소개

## 4. 데모 준비 체크리스트

### 4.1 데이터 준비

- [ ] 테스트 계정 로그인 상태
- [ ] 샘플 스터디 2~3개 생성 (진행 중인 스터디)
- [ ] 채팅 메시지, 핀 고정 메시지 준비
- [ ] 자료실에 파일 3~4개 업로드
- [ ] 이전 미팅 AI 요약 결과 1개 이상
- [ ] 퀴즈 기록 (오답 노트, 약점 분석 데이터)
- [ ] 캘린더에 일정 등록

### 4.2 환경 준비

- [ ] 네트워크 상태 확인 (와이파이 안정성)
- [ ] 브라우저 확대율 125~150%
- [ ] 앱 화면 미러링 준비 (scrcpy 또는 Vysor)
- [ ] 안드로이드 폰 2대 충전 및 블루투스 켜짐 확인
- [ ] 네트워크 불안정 대비 녹화 영상/GIF 백업

### 4.3 리허설

- [ ] 전체 플로우 최소 3회 연습
- [ ] 타이머로 10분 내 종료 확인
- [ ] AI 커리큘럼 생성 데모용 주제 미리 테스트
- [ ] 블루투스 출석 거리 테스트
