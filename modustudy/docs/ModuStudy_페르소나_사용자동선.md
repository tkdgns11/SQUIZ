## 2. 핵심 사용자 동선 (User Journey)

### Journey 1: 스터디 찾기 → 가입 (이지은)

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    스터디 참여자의 여정 (이지은)                           │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  [1. 프로필]        [2. AI 추천]        [3. 신청]         [4. 활동]      │
│                                                                          │
│  ┌─────────┐      ┌──────────┐       ┌──────────┐      ┌──────────┐    │
│  │ 매칭    │      │ 🤖 AI가  │       │ 상세보기 │      │ 팀 활동  │    │
│  │ 프로필  │─────▶│ 추천     │──────▶│ + 신청   │─────▶│ 시작     │    │
│  │ 작성    │      │          │       │          │      │          │    │
│  └─────────┘      └──────────┘       └──────────┘      └──────────┘    │
│       │                │                   │                 │          │
│       ▼                ▼                   ▼                 ▼          │
│  관심 카테고리    "이 스터디 추천!"     매칭 점수 85점     승인 알림 →    │
│  가능 시간대      "일정 80% 일치"      추천 이유 확인      팀 입장        │
│  실력 수준 입력   "레벨도 딱 맞아요"   자기소개 작성                      │
│                                                                          │
└─────────────────────────────────────────────────────────────────────────┘

**상세 흐름:**

| 단계 | 화면 | 사용자 행동 | 시스템 반응 |
|-----|------|------------|------------|
| 1 | 마이페이지 | 매칭 프로필 작성 (카테고리/시간/레벨) | 프로필 저장, 추천 활성화 |
| 2 | 스터디 탐색 | "🤖 AI 추천" 탭 클릭 | **매칭 점수 기반 스터디 목록 표시** |
| 3 | 스터디 목록 | 추천 스터디 클릭 | 상세 페이지 + **추천 이유 표시** |
| 4 | 스터디 상세 | "일정 80% 일치, 레벨 적합" 확인 | 매칭 점수 상세 breakdown |
| 5 | 스터디 상세 | "가입 신청" 클릭 | 신청 모달 표시 |
| 6 | 신청 모달 | 자기소개 작성 후 제출 | "신청 완료" 알림 |
| 7 | - | (대기) | 팀장이 승인 시 "승인됨" 알림 |
| 8 | 알림 | 알림 클릭 | 팀 메인 화면으로 이동 |

### Journey 2: 스터디 모집 → 운영 (김민수)

```
┌─────────────────────────────────────────────────────────────────────────┐
│                    스터디 리더의 여정 (김민수)                            │
├─────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  [1. 생성]      [2. 모집]       [3. 일정 조율]      [4. 운영]            │
│                                                                          │
│  ┌─────────┐  ┌──────────┐    ┌──────────┐       ┌──────────┐          │
│  │ 팀 생성 │  │ 신청자   │    │ 🤖 AI    │       │ 팀 운영  │          │
│  │         │─▶│ 심사     │───▶│ 일정추천 │──────▶│ 시작     │          │
│  └─────────┘  └──────────┘    └──────────┘       └──────────┘          │
│       │            │               │                  │                 │
│       ▼            ▼               ▼                  ▼                 │
│  팀 이름       자기소개 +      설문 배포 →         최적 시간으로         │
│  유형 선택     매칭점수 확인   "화 20시 추천!"     정기 미팅 설정        │
│                                "4명 가능"                                │
└─────────────────────────────────────────────────────────────────────────┘

**상세 흐름:**

| 단계 | 화면 | 사용자 행동 | 시스템 반응 |
|-----|------|------------|------------|
| 1 | 홈 | "+ 팀 만들기" 클릭 | 팀 생성 모달 표시 |
| 2 | 팀 생성 모달 | 팀명, 유형(알고리즘), 설명 입력 | 팀 생성 + 기본 채널 자동 생성 |
| 3 | 팀 설정 | "모집글 작성" 클릭 | 모집글 공개 (스터디 탐색에 노출) |
| 4 | - | (대기) | 신청자 발생 시 알림 |
| 5 | 신청 관리 | 신청자 자기소개 + **매칭점수** 확인 | 승인/거절 |
| 6 | 팀 설정 | **"일정 조율" 클릭** | 일정 설문 생성 모달 표시 |
| 7 | 설문 생성 | 후보 시간대 선택 (화 20시, 수 21시...) | 팀원에게 설문 알림 발송 |
| 8 | - | (팀원들이 응답) | 응답 현황 실시간 업데이트 |
| 9 | 설문 결과 | **"🤖 AI 추천 시간"** 확인 | **최적 시간 Top 3 표시** |
| 10 | 팀 설정 | 추천 시간으로 정기 미팅 설정 | 캘린더에 반복 일정 등록 |

## 4. MVP 우선순위별 동선 (AI 기능 포함)

### P0 (필수) - 핵심 기능

```
[회원가입/로그인] → [스터디 탐색] → [가입 신청] → [팀 입장]
                                         ↓
[팀 생성] → [초대 링크] → [팀원 모집] → [팀 운영]
                                         ↓
                         [음성 채널] → [미팅 시작] → [🤖 AI 요약]
```

### P1 (중요) - AI 추천/예측

```
[🤖 AI 스터디 추천] → [매칭 점수 확인] → [가입 신청]
         ↓
[데일리 스탠드업] → [🖐️ 손들기 제스처] → [발언 대기열] → [AI 요약]
         ↓
[🤖 일정 조율] → [설문 배포] → [AI 추천 시간] → [정기 미팅 설정]
```

### P2 (확장) - 예측 & 관리

```
[대시보드] → [🤖 참여 예측] → [위험 팀원 감지] → [개선 제안]
         ↓
[회고 생성] → [AI 초안] → [KPT 작성] → [Try → 이슈 전환]
         ↓
[이슈 관리] → [칸반 보드] → [담당자 지정]
```
