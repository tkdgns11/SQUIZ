# SSAFY 보고서 API (Report)

## 기본 정보
- Base URL: `/api/v1/studies/{studyId}/reports`
- 인증: JWT 필요 (스터디장)

---

## 엔드포인트 목록

| Method | Endpoint | 설명 | 인증 |
|--------|----------|------|------|
| GET | `/` | 보고서 목록 조회 | O |
| GET | `/{reportId}` | 보고서 상세 조회 | O |
| POST | `/application` | 신청서 생성 | O |
| POST | `/monthly` | 결과보고서 생성 | O |
| PUT | `/{reportId}` | 보고서 수정 | O |
| GET | `/photos` | 활동사진 갤러리 | O |
| PUT | `/photos/select` | 보고서용 사진 선택 | O |

---

## API 상세

### 1. 보고서 목록 조회

**Request**
```
GET /api/v1/studies/{studyId}/reports?type=MONTHLY_RESULT
Authorization: Bearer {accessToken}
```

| Parameter | Type | 필수 | 설명 |
|-----------|------|------|------|
| type | string | X | APPLICATION / MONTHLY_RESULT |

**Response**

### 3. 신청서 생성

**Request**
```
POST /api/v1/studies/{studyId}/reports/application
Authorization: Bearer {accessToken}
Content-Type: application/json
```
```json
{
  "goal": "알고리즘 문제 해결 능력 향상",
  "plan": "매주 2회 온라인 모임, 백준 문제 풀이 및 코드 리뷰"
}
```

> 스터디 정보, 멤버 정보는 시스템에서 자동 입력됩니다.

**Response**
```json
{
  "success": true,
  "data": {
    "id": 1,
    "reportType": "APPLICATION",
    "title": "알고리즘 스터디 신청서",
    "status": "DRAFT",
    "message": "신청서가 생성되었습니다. 미리보기 후 제출해주세요."
  }
}
```

### 4. 결과보고서 생성

**Request**
```
POST /api/v1/studies/{studyId}/reports/monthly
Authorization: Bearer {accessToken}
Content-Type: application/json
```
```json
{
  "reportMonth": "2025-01",
  "achievements": "팀원 모두 백준 실버 달성",
  "nextPlan": "2월 목표: 골드 문제 도전",
  "photoIds": [1, 2, 3]
}
```

> 출석 현황, 활동 내역은 시스템에서 자동 집계됩니다.

**Response**
```json
{
  "success": true,
  "data": {
    "id": 2,
    "reportType": "MONTHLY_RESULT",
    "title": "2025년 1월 결과보고서",
    "status": "DRAFT",
    "autoGeneratedData": {
      "totalSessions": 4,
      "averageAttendance": 92.5,
      "totalMeetingMinutes": 360
    },
    "message": "결과보고서가 생성되었습니다. 내용을 확인하고 수정해주세요."
  }
}
```

---

### 5. 보고서 수정

**Request**
```
PUT /api/v1/studies/{studyId}/reports/{reportId}
Authorization: Bearer {accessToken}
Content-Type: application/json
```
```json
{
  "content": {
    "goal": "수정된 목표",
    "plan": "수정된 계획"
  },
  "status": "SUBMITTED"
}
```

| status | 설명 |
|--------|------|
| DRAFT | 초안 |
| SUBMITTED | 제출 완료 |

**Response**
```json
{
  "success": true,
  "message": "보고서가 수정되었습니다."
}
```

### 7. 보고서용 사진 선택

**Request**
```
PUT /api/v1/studies/{studyId}/reports/photos/select
Authorization: Bearer {accessToken}
Content-Type: application/json
```
```json
{
  "photoIds": [1, 3, 5]
}
```

```json
{
  "success": true,
  "message": "보고서용 사진이 선택되었습니다.",
  "data": {
    "selectedCount": 3
  }
}
```

## 에러 코드

| 코드 | 설명 |
|------|------|
| REPORT_NOT_FOUND | 보고서를 찾을 수 없음 |
| NOT_STUDY_LEADER | 스터디장 권한 필요 |
| DUPLICATE_REPORT | 이미 해당 월 보고서 존재 |
| CANNOT_EDIT_SUBMITTED | 제출된 보고서는 수정 불가 |
