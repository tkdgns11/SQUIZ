1. 미팅 AI (STT/요약) - 핵심 기능
================================================================================
기능: 미팅 녹음 후 텍스트 변환 및 자동 요약 (실시간 자막 X)

[핵심 기능]
- U-FR-073: 미팅 STT - 미팅 종료 후 녹음 파일을 텍스트로 변환
- U-FR-076: 미팅 요약 - 미팅 종료 후 전체 대화 내용 요약
- U-FR-078: 액션 아이템 추출 - 해야 할 일 자동 추출

- U-FR-074: 화자 분리 - WebRTC 녹음 시 참가자별 자동 분리 (pyannote 불필요)
- U-FR-077: 발언자별 요약 - 각 참가자가 말한 내용 개별 요약

[사용 기술]
- Speech-to-Text (STT): Whisper medium (faster-whisper, GPU 추론)
- LLM 요약: Qwen3-8B (LoRA 파인튜닝 완료, GGUF Q4_K_M 양자화)
- 화자 분리: WebRTC 녹음 시 자동 분리 (별도 모델 불필요)

================================================================================

** 상용 API (GPT-4/Claude) 사용 안 함 - 자체 추론 서버 운영 **

[상용 API 미사용 이유: 데이터 프라이버시]
--------------------------------------------------------------------------------
스터디 녹음 데이터 = 민감한 개인 정보

- 상용 API 사용 시: 사용자 데이터가 OpenAI/Anthropic 서버로 전송됨
- 자체 호스팅 시: 데이터가 우리 서버에만 존재, 외부 유출 없음

포함되는 민감 데이터:
- 스터디 미팅 녹음 내용
- 참가자 음성/발언
- 회의 중 논의된 개인정보, 프로젝트 정보

→ 사용자 데이터 보호를 위해 오픈소스 모델 자체 호스팅 필수

6. 학습 정보
================================================================================

[Qwen3-8B 요약 파인튜닝 (LoRA)]
--------------------------------------------------------------------------------
- 모델: Qwen/Qwen3-8B
- 목적: IT 스터디 회의록 요약 특화
- 데이터: GPT-4o-mini로 생성한 합성 데이터 700개
  * Type A (일반 요약): 500개 - 회의록 → 요약
  * Type B (파트 요약): 150개 - 긴 회의 청크 → 파트별 요약
  * Type C (통합 요약): 50개 - 파트 요약들 → 최종 요약
  * Train: 630개 / Validation: 70개
- 학습 환경: SSAFY L40S 46GB GPU
- 학습 설정:
  * QLoRA (4-bit 양자화, BitsAndBytes)
  * LoRA rank=64, alpha=128
  * MAX_SEQ_LENGTH=4096
  * Epochs: 4, 316 steps
- 결과:
  * LoRA 어댑터 병합 완료
  * GGUF Q4_K_M 양자화 완료 (4.7GB)
