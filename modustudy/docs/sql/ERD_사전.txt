ERD 사전 - Board 도메인

[board_post]
- id (BIGINT, PK) : 모집글 ID
- created_at (DATETIME) : 생성일시
- updated_at (DATETIME) : 수정일시
- user_id (BIGINT, FK -> user.id) : 작성자 ID
- study_id (BIGINT, FK -> study.id) : 스터디 ID
- category (VARCHAR(20)) : 게시글 카테고리 (예: FREE)
- title (VARCHAR(200)) : 모집글 제목
- content (TEXT) : 모집글 내용
- view_count (INT) : 조회수
- like_count (INT) : 좋아요 수
- comment_count (INT) : 댓글 수
- is_deleted (TINYINT) : 삭제 여부 (0: 미삭제, 1: 삭제)

[board_comment]
- id (BIGINT, PK) : 댓글 ID
- created_at (DATETIME) : 생성일시
- updated_at (DATETIME) : 수정일시
- post_id (BIGINT, FK -> board_post.id) : 모집글 ID
- user_id (BIGINT, FK -> user.id) : 작성자 ID
- parent_id (BIGINT, FK -> board_comment.id, NULL 가능) : 부모 댓글 ID
- content (TEXT) : 댓글 내용
- is_deleted (TINYINT) : 삭제 여부 (0: 미삭제, 1: 삭제)

[board_like]
- id (BIGINT, PK) : 좋아요 ID
- created_at (DATETIME) : 생성일시
- updated_at (DATETIME) : 수정일시
- post_id (BIGINT, FK -> board_post.id) : 모집글 ID
- user_id (BIGINT, FK -> user.id) : 사용자 ID
